# 冒泡排序

```java
//Main.java文件代码：

public class Main {
    public static void main(String[] args) {
        int[] arrys= {2,1,3,4,5,6,3,1,7,676,254,546,654};
        //冒泡排序
        for (int i= arrys.length; i> 1; i--){
            for (int j=0; j< i-1; j++){
                if (arrys[j]> arrys[j+1]) {
                    int temp = arrys[j];
                    arrys[j] = arrys[j + 1];
                    arrys[j + 1] = temp;
                }
            }
        }
		//遍历数组
        for (int i= 0; i< arrys.length; i++){
            System.out.println(arrys[i]);
        }
    }
}

```
