# 选择排序

```java
//Main.java文件代码

public class Main {

    public static void main(String[] args) {
        int[] array= {2,1,3,4,5,6,3,1,7,0};
        //选择排序
        for (int i= 0; i<array.length-1; i++){
            int min= i;
            for (int j= i+ 1; j< array.length; j++){
                if (array[j] < array[min]){
                    min= j;
                }
            }
            if(min != i){
                int temp;
                temp= array[min];
                array[min]= array[i];
                array[i]= temp;
            }
        }
        //遍历数组
        for (int i= 0; i< array.length; i++){
            System.out.println(array[i]);
        }
    }
}

```
