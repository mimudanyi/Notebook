# 转向语句

## 一、break

- 用在switch语句中，终止switch语句，防止case穿透现象。
- 用在循坏语句（for、while、do…while）中，终止离它最近的循环语句。

```java
for(int i=1; i<10; i++){
if(i==5){
break;
}
System.out.println("i="+i);
}
a:for(int k=0; k<2; k++){
b:for(int i=0; i<10; i++){
if(i==5){
break a;
}
System.out.println("i==>"+i);
}
}
```

## 二、continue

在循环中，终止本次循坏，直接跳到下一个循坏。
