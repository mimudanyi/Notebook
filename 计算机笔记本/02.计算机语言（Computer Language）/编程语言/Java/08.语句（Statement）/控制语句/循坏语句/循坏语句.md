# 循坏语句

## 一、for语句

```java
for ( 初始化表达式 ; 循环条件布尔表达式 ; 循环计数器表达式 ){
		//code
}
```

## 二、for each语句

```java
for ( int n ： 变量){
		//code
}
```

此循环可以更简单地遍历数组如：

```java
public class Main {

		public static void main ( String [ ] args ) {

				int [ ] ns = { 1, 4, 9, 16, 25 } ;

				for ( int n : ns ) {
						System.out.println ( n ) ;
				}
		}
}
```

## 三、while语句

```java
while ( 布尔表达式 ){
		//code
}
```

## 四、do…while语句

```java
do{
		//code
}while ( 布尔表达式 )；
```
