# 资源绑定器

## 一、用来配置配置文件

当配置文件中的内容格式是：

```properties
# 这是一个注释
key1=value
key2=value
```

Java 规范中有要求：属性配置文件以.properties结尾，但这不是必须的。

其中Properties是专门存放属性配置文件内容的一个类。经常变化的信息建议放在属性配置文件里。

资源绑定器（最终方式，没有比这个更简单了）

资源绑定器绑定的文件必须放在类路径下或者Maven的资源文件夹中

！！！！！重点！！！！！！！

```java
//App.java文件代码：

import java.util.ResourceBundle;

public class App {
    public static void main(String[] args) {
        ResourceBundle resourceBundle = ResourceBundle.getBundle("info");
        String keyName = resourceBundle.getString("key1");
        System.out.println(keyName);
    }
}

//info.properties文件代码:

# 配置文件注释
key1=1
```

> 运行结果：
>
> ```
> 1
> ```
