# b标识符

## 一、定义

所有的组成部分都需要名字。类名、变量名以及方法名都被称为标识符。

## 二、命名规则

- 标识符都可以以英文字母（A-Z 或者 a-z）、美元符（$）或者下划线（_）开头；
- 首字符之后可以是英文字母（A-Z 或者 a-z）、美元符（$）、下划线（_）或数字的任何字符组合；
- 关键字不能用作标识符；
- Java标识符区分大小写，Java虽然严格区分大小写，但是对于类名来说。如果一个Java源文件同时出现了大小写的类（如helloworld和HelloWorld）,那么谁在前面就生成谁，所以最后不要让类名“相同”；
- 标识符理论上是没有长度限制的。
  - 合法标识符举例：age、$salary、_value、__1_value
  - 非法标识符举例：123abc、-salary

## 三、命名规范

- 见名知意；
- 遵循驼峰命名方式；
- 类名和接口名每个单词首字母大写；
- 方法名和变量名首个单词全部小写，后面每个单词首字母大写；
- 常量名全部大写，单词和单词之间用下划线衔接。
