# HTML 标签和属性

## 文档

```html
<html>文档</html>
```

## 主体

```html
<body>主体</body>
```

## 标题

```html
<h1>标题1</h1>
<h2>标题2</h2>
<h3>标题3</h3>
<h4>标题4</h4>
<h5>标题5</h5>
<h6>标题6</h6>

<!--属性-->
	<h1 align="center"><!--拥有关于对齐方式的附加信息-->
        
```

## 段落

```html
<p>段落</p>
```

## 折行（换行）

```html
<br/>
```

## 链接

```html
<a href="http://www.w3school.com.cn">This is a link</a>
```

## 图像

```html
<img src="w3school.jpg" width="104" height="142" />
```

## 注释

```html
<!--注释-->
```

## 水平线

```html
<hr/>
```
