# 循坏语句

## 一、for语句

```js
for ( 【初始化表达式】 ; 【循环条件布尔表达式】 ; 【循环计数器表达式】 ){
		//code
}
```

## 二、for in语句

在JavaScript中，for in循环不仅可以遍历对象的属性，还可以遍历数组。

```js
for ( var 【变量名】 in 【数组名】){	//其中变量名为数组下标
		//code
}
```

此循环可以更简单地遍历数组如：

```js
var person={fname:"John",lname:"Doe",age:25};

for (x in person){
  txt=txt + person[x];
}
```

## 三、while语句

```js
while ( 【布尔表达式】 ){
		//code
}
```

## 四、do…while语句

```js
do{
		//code
}while ( 【布尔表达式】 )；
```
