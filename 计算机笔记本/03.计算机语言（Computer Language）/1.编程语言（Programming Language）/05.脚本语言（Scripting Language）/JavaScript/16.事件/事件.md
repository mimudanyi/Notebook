# 事件

事件是可以被 JavaScript 侦测到的行为。

HTML 事件是发生在 HTML 元素上的事情。

当在 HTML 页面中使用 JavaScript 时， JavaScript 可以触发这些事件。

JavaScript 事件：

- blur 失去焦点
- change 下拉列表选中项改变，或文本框内容改变
- click 鼠标点击
- dblclick 鼠标双击
- focus 获得焦点
- keydown 键盘按下
- keyup 键盘弹起
- mousedown 鼠标按下
- mouseover 鼠标经过
- mouseout 鼠标离开
- mouseup 鼠标弹起
- reset 表单重置
- select 文本被选中
- submit 表单提交

注册事件的两种方式

- 第一种：在标签中使用事件句柄：如 onclick
- 第二种：使用纯 JS 代码完成事件的注册
  - 第一步：先获取这个按钮的对象；
    var btnOj = document.getElementById("【id名】")
  - 第二步：给按钮对象的onclick属性赋值；
    btnOj.onclick = 回调函数 （回调函数不能加()，只写函数名）
