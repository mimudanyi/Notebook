# 条件语句

## 一、if语句：

```java
if ( 布尔表达式 ){
		//code;
}
```

## 二、if else语句（可以不断嵌套）：

### 2.1 形式一

**如果只有两个选择，也可以换成三目运算符**

```java
if ( 布尔表达式 ){
		//code;
}else{
		//code
}
```

```java
 (condition1) ? ture-doing : else-doing; 
```

### 2.2 形式二

```java
if ( 布尔表达式 ){
		//code;
}else if( 布尔表达式 ){
		//code;
}else if( 布尔表达式 ){
	//code;
}else if( 布尔表达式 ){
//code;
}
```

### 2.3 形式三

```java
if ( 布尔表达式 ){
		//code;
}else if( 布尔表达式 ){
		//code;
}else if( 布尔表达式 ){
		//code;
}else if( 布尔表达式 ){
		//code;
}else{
		//code;
}
```

## 三、swich case语句：

```java
switch ( 变量值 )
{
		case 变量值1：//code；break；
		case 变量值2：//code；break；
		case 变量值3：//code；break；
		case 变量值4：//code；break；
		case 变量值5：//code；break；
		case 变量值6：//code；break；
        default://code；
}
```

## 注意

- 其中的break语句和default语句不是必须的；

- JDK8之前其中的值只支持int类型，之后支持int和String类型；

- 本质上只支持int和String,但是byte、short、char也可以使用，因为可以进行动类型转换；

- 如果没有break，就会发生case穿透现象，直到遇到break才会停止。

- 可以进行case合并：

  ```java
  switch ( 变量值 ){
  		case 变量值1：case 变量值2：case 变量值3：//code；break；
  		case 变量值4：case 变量值5：//code；break；
  		case 变量值6：//code；break；
  		default；//code；
  }
  ```
