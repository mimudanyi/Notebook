## 数据类型的转换

- 数据类型转换必须满足如下规则：

  - Java中八种数据类型中，除不能对boolean类型进行类型转换，剩下的八种之间都可以转换。

  - 在把容量大的类型转换为容量小的类型时必须使用强制类型转换。

  - 转换过程中可能导致溢出或损失精度因为 byte 类型是 8 位，最大值为127，所以当  int  强制转换为 byte 类型时，值 128 时候就会导致溢出，例如： `int i =128; byte b = (byte)i`。

  - 浮点数到整数的转换是通过舍弃小数得到，而不是四舍五入，例如：

    `(int)23.7 == 23;`

    `(int)-45.89f == -45;`

- 自动转换（小到大）：

  - 整型、实型（常量）、字符型数据可以混合运算。运算中，不同类型的数据先转化为容量大的那一种一类型，然后进行运算。转换从低级到高级：

低 ------------------------------------> 高

byte,short,char—> int —> long—> float —> double

- byte、short和char三种类型做符合运算时，先转换成int类型再做运算。
- **强制转换（大到小）：**

```
long x = 100L;
int y = (int)x;
```

转换过程中可能导致溢出或损失精度因为 byte 类型是 8 位，最大值为127，所以当 int 强制转换为 byte 类型时，值 128 时候就会导致溢出，例如：int i =128;  byte b = (byte)i;

## 注意

- 在Java中，整数型字面量（数据）默认当成int类型，当给变量赋值是，需要在字面量后面添加L/l，如long=1342L;
- 任意一个浮点型都比整数型空间大。
- Java规定：任意一个浮点型数据都被当做double来处理，如果想让这个浮点数被当做float类型来处理，那么在后面添加f/F。
