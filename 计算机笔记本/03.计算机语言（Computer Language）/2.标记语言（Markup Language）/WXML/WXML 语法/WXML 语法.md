# WXML 语法

## 一、标签

```html
<标签名 属性名="属性名1" 属性名="属性名2">
    ...
</标签名>
```

## 二、属性

| 属性名         | 类型         | 描述           | 注解                                   |
| :------------- | ------------ | -------------- | -------------------------------------- |
| id             | String       | 组建的唯一标示 | 保持整个页面唯一                       |
| class          | Strin        | 组件的样式类   | 在对应的WXSS中定义样式类               |
| style          | String       | 组件的内联样式 | 可以动态设置的内联样式                 |
| hidden         | Boolean      | 组件是否显示   | 所有组件默认显示                       |
| data-*         | Any          | 自定义属性     | 组件上触发的事件时，会发送事件处理函数 |
| bind* / catch* | EventHandler | 组件的事件     | 详见事件                               |
|                |              |                |                                        |
