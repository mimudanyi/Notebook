# 处理器（Central Processor Unit,CPU）

## 一、中央处理器的组成

- 运算器（Arithmetic Logic     Unit，ALU，数据通路Datapath）：负责运算从内存中读入寄存器的数据。
- 控制器（Control     Unit,CU）：负责把内存上的指令、数据读入寄存器，并根据指令的结果控制计算机。
- 时钟：负责运算从内存中读入寄存器的数据。
- 寄存器：可以用来暂存指令、数据和地址。可以将其看作是内存的一种。根据种类不同，一个CPU内部会有20-100个寄存器。

## 二、CPU指令执行过程

- 取指令阶段
  是将内存中的指令读取到 CPU 中寄存器的过程，程序寄存器用于存储下一条指令所在的地址

- 取指令阶段
  是将内存中的指令读取到 CPU 中寄存器的过程，程序寄存器用于存储下一条指令所在的地址

- 执行指令阶段
  译码完成后，就需要执行这一条指令了，此阶段的任务是完成指令所规定的各种操作，具体实现指令的功能。

- 访问取数阶段
  根据指令的需要，有可能需要从内存中提取数据，此阶段的任务是：根据指令地址码，得到操作数在主存中的地址，并从主存中读取该操作数用于运算。

- 结果写回阶段（Write Back，WB）
  把执行指令阶段的运行结果数据“写回”到某种存储形式：结果数据经常被写到CPU的内部寄存器中，以便被后续的指令快速地存取；
