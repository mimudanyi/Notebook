# 计算机组成概述

##一、硬件 

由中央处理机 (运算器和控制器) 、存储器、输入设备和输出设备等部件组成，它构成了系统本身和用户作业赖以活动的物质基础和工作环境。计算机基本功能（冯·诺依曼体系结构）：

### （一）、处理——中央处理器(Central Processor Unit,CPU)

1.控制器(Control Unit,CU)

2.运算器(Arithmetic Logic Unit，ALU，数据通路Datapath)

①.图形处理单元（Graphic Processor Unit,GPU）

3.时钟

4.寄存器

### （二）、存储——储存器（Memory）

1.内存储器（Memory）:由动态随机访问存储器（Dynameic Random Access Memory,DRAM)芯片组成

①.缓冲器

②.高速缓存（Cache Memory）:由动态随机访问存储器（Static Random Access Memory,SRAM）芯片组成

2.外存储器：

（四）、输入——输入设备（Input Device）

（五）、输出——输出设备（Output Device）

（六）、输入输出控制总线

1.单总线

2.多总线
